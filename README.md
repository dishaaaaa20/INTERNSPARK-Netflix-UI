# Netflix Clone

A responsive Netflix-inspired web interface built using HTML, CSS, and JavaScript.
This project recreates the core layout and visual style of the Netflix homepage while keeping the code beginner-friendly and easy to understand.

---

## Overview

The project includes:

* A responsive navigation bar
* Featured hero banner section
* Horizontally scrollable movie rows
* Interactive movie cards
* Hover animations
* Multiple content categories
* Clean UI inspired by Netflix

The goal of this project was to practice frontend development concepts such as layout design, styling, DOM manipulation, and basic interactivity.
